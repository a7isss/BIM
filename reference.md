# 
      
        Revit 2026 Tutorial - Complete Floor Plan 🏠

We have prepared a simple template that you can use to create your own floor plan. It contains wall compositions and components to help you out.

1- Set Up Levels Height
-----------------------

Even if you don’t need elevations or 3D views at the moment, you still need to assign the walls to specific levels. Go to an elevation view to ensure all the required levels are created. Set them to the correct height, although you can adjust them later on.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/a0e3c-a45-732f-c21b-e7136d204df2_04_image-asset.png)

2- Place Walls Intersection at Internal Origin
----------------------------------------------

Although you might not care about the coordinate system, you should still place one of the corners of the building at the intersection of the **Internal Origin** of the project. The internal origin is indicated by the arrows symbol. The internal origin is not to be confused with the Project Base Point and Survey Point, although all these points should be in the same position when you start.

If you export to CAD, this point will be used as the origin. Same thing if you link CAD or another format.

If you are confused about Revit’s coordinate system, make sure to check out our popular guide over here: [13 Tips to Understand Revit Base Points and Coordinate System](https://bimpure.com/blog/13-tips-to-understand-revit-base-points-and-coordinate-system)

Use shortcut VG to go to the Visibility/Graphics menu. Scroll down to find the site submenu. Check the **Internal Origin** box. The arrow symbol should appear in your plan view.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/ceb16-5d48-550-034a-d672c1bbef1_05_image-asset.png)

Create walls at the intersection of the origin.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/f4de4b-f64-6040-bbbb-0aae470a07e2_06_image-asset.png)

3- Optional: Create a Layout With Detail Lines
----------------------------------------------

One of the most common complaints from AutoCAD users is that there is no way to create a quick layout in Revit. That’s not quite right, you just need to use the proper tools.

You can create a layout using walls directly, but you can also create a layout using detail lines. Use shortcut **DL**. These lines are only visible in a single view and are not part of the 3D model. You should use coloured lines to distinguish from the model elements. In the example below, we use red lines.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/48323e4-4e5e-f558-e13-56c62bbabdc6_07_image-asset.png)

After creating a draft layout, you can match your walls and other elements to the detail ones. The best way is to use the **Align** tool (shortcut: **AL**). Type AL, click the detail line reference, then click the edge of the wall. The wall will move to be aligned with the detail line.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/0f681d-8ab-2d0-2e4e-1d021c2a7b04_08_image-asset.png)

4- Use Shortcut CS to Create Similar
------------------------------------

Want to save some time? Select an element and use shortcut **CS** to create a similar one. This is one of the best ways to be efficient. In the example below, we use this tool to create walls quickly.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/453fbd-2ba-088-3117-aedbd55dc32_09_image-asset.png)

5- Use Temporary Dimensions
---------------------------

Select an element. You will see a dimension in blue. This is called a temporary dimension. Drag the dots to adjust the witness lines. Change the blue text value to adjust the dimension. Click on the dimension symbol to make the dimension permanent in the view.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/6de31de-c33-0a70-0d7e-7c8f7812406c_10_rp-temp-dimensions.png)

6- Use Trim to Adjust Walls Quickly
-----------------------------------

Former AutoCAD users might be familiar with the **TRIM** (shortcut: **TR**) tool. Use it to **Trim** or **Extend** elements to be joined together.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/08ff27-24db-b81c-8f2-108e0a84bd5b_11_image-asset.png)

The same TRIM tool can also be used to extend elements. In the example below, we join two walls.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/141d6ce-3ab-6a05-54-fe62fe317c85_12_image-asset.png)

After using these tools, you complete the layout:

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/472c5e-2c71-1b35-0bb1-f62fa4ab4ea4_13_image-asset.png)

7- Add Windows and Doors, Adjust Family Types
---------------------------------------------

Once your wall layout is complete, you can start to add doors and windows. Using shortcuts is a great way to save time:

**DR** - Door

**WN** \- Windows

**WA** - Walls

**Spacebar** - Flip door, walls, and windows orientation.

In the example below, we use both the standard door and pocket door families. Make sure to use the correct type in the type selector.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/58e733-1b68-a677-66e4-4f85366de77d_14_image-asset.png)

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/d24cc4c-f00a-53fc-f8dc-8e12deef77e7_15_image-asset.png)

Doors are easy to create and control. Click the arrow symbol or hit **SPACEBAR** to flip the door.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/0125733-032d-7242-06fd-ff772065e815_16_rp-flip-door.png)

You can start adding windows. Click on the wall, then make sure the side is properly placed. The arrows symbol is located on the exterior side of the window, so make sure to flip it if required.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/587f42b-eff-0aef-e71b-1224e8ee28b_17_image-asset.png)

Create a type of window for each size you need. Adjust the dimensions to match your needs.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/bc322e3-c60b-67ea-02d1-bc3b168704_18_image-asset.png)

Make sure to adjust the window sill height as in the example below. It is located in the instance properties.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/e35-5b-8a84-6e-2708fbba1ae_19_image-asset.png)

Our layout now includes walls, doors, and windows! Ready for the next step?

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/38204ec-4623-caeb-bd12-6db75e0247fa_20_image-asset.png)

8- Add Components and Lines
---------------------------

This is the part where many people might struggle. Finding good components and families in Revit is difficult.

Our free floor plan template can help. [Download](https://bimpure.com/free-template) it for free.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/3df272-4853-1055-430-5f66ef1a66_image.png)

Most of these components are **2D families**. That's great for a floor plan, but you might want to eventually upgrade to 3D families.

Start adding components provided in the template. Use the **Create Similar** tool (shortcut CS) or copy/paste. Below, we start adding a toilet, fridge, wardrobe, etc.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/70e5c3d-acea-5d11-135d-af4c1338401_21_image-asset.png)

If you can't find families, you can use the Model Lines tool (shortcut: LI) to draw elements such as the kitchen.

Each component family might work differently. For example, the toilet family has to be hosted on a wall; it can’t just be floating around.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/21d5f20-232b-3b7b-133-8cb8dc1ae0fd_22_image-asset.png)The wardrobe rod family has arrows to control the length of each instance.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/65f7170-a0ec-065e-161b-04c317b4af5_23_rp-drag-arrows.png)

Take time to look at each family to understand how they work.

9- Use Overlay to Visualize Other Floors
----------------------------------------

You're ready to work on the second floor. But upon opening the view, you can't see anything.

It’s always helpful to see what’s going on below. In the instance of the second-floor view, activate the **Underlay** and set the Base Level to Level 1. The level 1 floor plan will appear in gray. The floor can be used as a reference.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/3a6a138-bf5f-0b63-7bbf-0e1a0156d35_24_image-asset.png)

10- Use Reference Planes to Align Components From One Floor to Another
----------------------------------------------------------------------

Another way to reference position on multiple levels is to use reference planes. These elements appear in dashed green lines and never print. In the example below, we add two reference planes where we plan to add a structural column. The planes are visible in all plan views.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/6d07120-18a7-8e7f-5a87-383dc2fc3d_25_image-asset.png)

11- Model Thin Floor for Finishes
---------------------------------

Your floor plan is starting to get quite complete! Let's add a ceramic floor pattern. Create a new floor using a thin type (12mm or 1/2’’) or use the type provided in the template.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/d6a2d00-5ffb-618a-e63e-b2aae3d85e0_26_image-asset.png)

12- Annotate the Drawing
------------------------

The template includes tags for windows and doors. The door tag indicate the **width**, while the windows tag displays the **width x height**.

To create tags, use shortcut **TG** and click on an element.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/8edb1c8-de3a-b2b-16fc-cf13c85a1b5_27_image-asset.png)

Tags are intelligent: if you modify the width of the door, the tag will automatically adjust.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/ac71801-af4e-bf02-d4d3-28851ad772d8_28_rp-tag-doors2.png)

You can switch the tag from horizontal to vertical in the options bar.

Use tags on all doors and windows. Then, add **dimensions** using the shortcut **DI**.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/21c2d3-10e1-e0c3-30b6-043552b55d0c_29_image-asset.png)

13- Create a Dark Wall Poche
----------------------------

Why not add a dark wall poche to create a nice presentation effect? Go to the Visibility/Graphics menu by using shortcut **VG**. Scroll down to the wall category. Click on **Cut Pattern**. In the background category, set a dark gray color (or black). Use the solid fill option.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/cccd7b3-147-c8c0-8bc6-dffc8c42e33_30_image-asset.png)

As you can see below, all walls in this view are affected by this setting.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/6b0626f-a283-5ad-50aa-5e3b8562fc_31_image-asset.png)

14- Place the Views on Sheets and Print
---------------------------------------

Make the final touch-ups, then place the view on a sheet. Adjust the crop region of the view around the house. Adjust information on the sheet. You are now ready to print! Good job.

![](https://kajabi-storefronts-production.kajabi-cdn.com/kajabi-storefronts-production/file-uploads/blogs/2147511612/images/a8a6320-6d81-4466-81dd-0c42a0ede20_32_image-asset.png)
